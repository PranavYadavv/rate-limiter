# rate-limiter

Distributed rate limiting with sliding window algorithm

## Installation

```bash
pip install rate-limiter
```

## Quick Start

```python
from rate_limiter import Rate
client = Rate()
client.run()
```

## Features

- High performance and memory efficient
- Comprehensive error handling
- Well tested with full coverage
- MIT licensed
